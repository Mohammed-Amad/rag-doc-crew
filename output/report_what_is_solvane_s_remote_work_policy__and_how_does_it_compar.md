# What is Solvane's remote work policy, and how does it compare to industry-standard hybrid policies?
## Executive Summary
Solvane's remote work policy is **role-based**, not seniority-based, with the signed job profile determining the default and a matrix setting company-wide limits. Key aspects of this policy include:
* **Role-based** policy determination
* Signed job profile sets the default remote work arrangement
* A company-wide matrix sets limits on remote work
For example, **Software & Fleet-Ops Eng.** have **3 max remote days/wk**, with onsite anchor days on **Tue, Wed**, and require manager approval.
